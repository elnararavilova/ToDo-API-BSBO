# ToDo лист API

## Описание
REST API для управления задачами с классификацией по матрице Эйзенхауэра.

## Технологии
- FastAPI
- Python 3.7+
- Временное хранилище данных

## Основные эндпоинты
- `GET /` — информация о приложении
- `GET /tasks` — список всех задач
- `GET /tasks/{task_id}` — задача по идентификатору
- `GET /tasks/quadrant/{quadrant}` — фильтрация по квадранту (Q1-Q4)
- `GET /tasks/status/{status}` — задачи по статусу выполнения
- `GET /tasks/search` — поиск по названию и описанию
- `GET /tasks/stats` — статистика по задачам

## Запуск приложения
1. Установите зависимости:  
   `pip install fastapi uvicorn`
2. Запустите сервер:  
   `uvicorn main:app --reload`
3. Перейдите: `http://127.0.0.1:8000/docs`
